# calculator
Простой веб-калькулятор. Реализует базовые арифметические операции, обработку десятичной точки, последовательные вычисления и корректную работу с кнопками очистки и равно.

## Стек

- **Frontend**: HTML, CSS, JavaScript (ES6+)

## Как запустить локально

1. Скачайте все три файла в одну папку:
   - `index.html`
   - `style.css`
   - `index.js`

2. Откройте файл `index.html` в любом современном браузере (двойным щелчком или через «Открыть с помощью»).
